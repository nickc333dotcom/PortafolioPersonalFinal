Proyecto: Portafolio Personal 
Para el funcionamiento del proyecto, hay que ir a la carpeta de client y ejectuar npm install. 
Hacer lo mismo con server.
Una vez instaladas las dependencias, iniciar el backend con: node index.js
                                    iniciar el frontend con: npm run dev
